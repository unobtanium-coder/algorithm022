以 leetcode#63 “不同路径2” 为例：

1、定义状态

    dp[i][j] = 从左上角(0,0)到(i,j)的路径条数

2、状态转移方程（含边界条件）

    if obstacleGrid[i][j] == 1:

        dp[i][j] = 0
    
    else:

        dp[i][j] = dp[i][j-1] + dp[i-1][j]
    
    (边界条件：需满足i-1/j-1>=0)

3、初始化

    m = len(obstacleGrid)

    n = len(obstacleGrid[0])

    dp[0][0] = 1 if obstacleGrid[0][0] == 0 else 0
